# 🎯 CS2 沙二 (Dust2) 道具教程管理器

一个纯前端的 CS2 道具点位管理工具。支持在高清地图上标记烟、闪、雷、火点位，附带多图教程与拖拽排序功能。

✨ **核心特性**：数据直接保存至本地文件系统（非浏览器缓存），无需后端，隐私安全。

👉 **[在线演示地址](https://zhuzh3378.github.io/CS2-Item-Memo/)**

## 📸 功能亮点

-   🗺️ **高清地图交互**：支持滚轮缩放、拖拽平移，精准标记点位坐标
-   📂 **文件系统直存**：基于 File System Access API，数据以 JSON 形式保存在你指定的本地文件夹中
-   🖼️ **结构化图片管理**：站位 / 出手 / 效果三桶分类，支持拖拽排序调整主图，详情页左右按钮切换 + 全屏查看
-   🔍 **高效检索**：支持按名称搜索、类型筛选、收藏过滤
-   💾 **导入导出**：一键备份/分享你的道具库（JSON 格式）
-   ⚠️ **防丢失保护**：编辑未保存时关闭页面会自动拦截提醒

## 🚀 快速开始

### 环境要求

-   **浏览器**：必须使用 **Chrome** 或 **Edge**（Firefox/Safari 不支持文件系统 API）
-   **运行方式**：推荐使用 [在线演示地址](https://zhuzh3378.github.io/CS2-Item-Memo/) 或本地服务器（HTTPS 环境）。直接双击打开 HTML 文件（`file://` 协议）可能导致存储功能不可用

### 使用步骤

1.  访问 [在线演示地址](https://zhuzh3378.github.io/CS2-Item-Memo/)
2.  首次使用时，点击左上角 **「📁 初始化存储目录」**，选择你电脑上的一个空文件夹作为数据存储位置
3.  切换到右上角 **「编辑模式」**
4.  在地图空白处 **右键点击** 即可添加道具点位
5.  填写名称、简介，上传站位/出手/效果截图（支持拖拽上传和拖拽排序）
6.  点击保存，数据将自动写入你选择的本地文件夹

### 本地部署（可选）

如果你希望离线使用或自定义地图：

1.  克隆本仓库或下载 `index.html` 和 `dust2.png`
2.  确保两个文件在同一目录下
3.  使用 VS Code Live Server 插件或 Python 启动本地服务器：
    ```bash
    # Python 3
    python -m http.server 8000
    ```
4.  访问 `http://localhost:8000` 即可使用

## 📁 数据结构说明

初始化存储目录后，工具会在该目录下自动生成：

```text
your-save-folder/
└── data.json       # 所有道具的元数据（名称、坐标、标签、Base64截图等）
```
> ⚠️ 注意: 当前版本截图以 Base64 嵌入 `data.json`。道具数量较多时文件体积会增大，建议定期使用「导出备份」功能归档。
